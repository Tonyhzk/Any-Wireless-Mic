## [1.0.1] - 2026-02-21

### 改进

- **跨平台配置目录** - 打包后配置文件存放到系统标准目录（Windows: `%APPDATA%`，macOS: `~/Library/Application Support`，Linux: `~/.config`）
- **Flask 3.1 兼容性** - 修复 Flask 3.1 与 Flask-SocketIO 的 session 属性兼容问题
- **版本号显示** - 窗口标题栏显示当前版本号
- **依赖版本锁定** - requirements.txt 固定所有依赖版本，补充 cryptography 依赖
- **打包配置** - 新增 macOS/Windows 打包 spec 文件、Inno Setup 安装脚本及安装包资源

## [1.0.0] - 2026-02-19

### 首次发布

- **实时音频传输** - 通过 WiFi 局域网将任意设备音频实时传输到电脑
- **虚拟麦克风** - 输出到 VB-Cable / BlackHole，兼容任意应用
- **自动重采样** - 自动适配不同设备的采样率
- **录制功能** - 本地与远程录制控制，WAV 格式输出
- **波形可视化** - 音频波形显示，支持点击跳转
- **实时波形** - 音频传输时实时显示音量波形
- **远程文件管理** - 手机端浏览、播放、下载、删除录音
- **二维码连接** - 扫码即连，零配置
- **自动 HTTPS** - 自动生成自签名证书
- **设备选择** - 音频输出设备选择与刷新
- **主题切换** - 暗黑/明亮主题支持
- **快捷键** - 空格播放/暂停，方向键快进/快退
- **安全删除** - 回收站或永久删除可选
- **模块化架构** - 清晰分离音频、服务器、界面、网页模块
